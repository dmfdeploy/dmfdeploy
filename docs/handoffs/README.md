# Handoffs

**Append-only lab notebook.** Newest file wins; older entries are historical context.
Never required reading to contribute.

Each file captures the operator's intent and state at the boundary between
agent sessions. Read the most recent one at boot to pick up where the prior
session left off. Older entries are preserved for provenance only.

> **Note:** Handoff files are exempt from the plans frontmatter convention.
> They are not tracking artifacts — they are session-context snapshots.
